bordes

Los bordes se pueden aplicar a cualquier elemento de HTML. Para ello utilizaremos la propiedad border y sus subpropiedades.

La propiedad border se compone de tres subpropiedades:

  • border-width: define el ancho del borde.
  • border-style: define el estilo del borde.
  • border-color: define el color del borde.

Para definir el ancho del borde podemos utilizar valores absolutos como px o valores relativos como em o rem. También podemos utilizar valores como thin, medium o thick.

Para definir el estilo del borde podemos utilizar valores como solid, dotted, dashed, double, groove, ridge, inset o outset.

Para definir el color del borde podemos utilizar cualquier valor de color que hemos visto anteriormente.

border: 1px solid black;

border: 5px solid black;

border: 1px solid red;

border: 1px solid #f00;

border: 1px solid rgb(255, 0, 0);

border: 1px solid rgba(255, 0, 0, 0.5);

border: 1px solid hsla(0, 100%, 50%, 0.5);

Todas las propiedades

Los bordes tienen muchas propiedades aqui la lista completa

  • border-width: define el ancho del borde.
  • border-style: define el estilo del borde.
  • border-color: define el color del borde.
  • border-top: define el borde superior.
  • border-top-width: define el ancho del borde superior.
  • border-top-style: define el estilo del borde superior.
  • border-top-color: define el color del borde superior.
  • border-right: define el borde derecho.
  • border-right-width: define el ancho del borde derecho.
  • border-right-style: define el estilo del borde derecho.
  • border-right-color: define el color del borde derecho.
  • border-bottom: define el borde inferior.
  • border-bottom-width: define el ancho del borde inferior.
  • border-bottom-style: define el estilo del borde inferior.
  • border-bottom-color: define el color del borde inferior.
  • border-left: define el borde izquierdo.
  • border-left-width: define el ancho del borde izquierdo.
  • border-left-style: define el estilo del borde izquierdo.
  • border-left-color: define el color del borde izquierdo.
  • border-radius: define el radio del borde.
  • border-top-left-radius: define el radio del borde superior izquierdo.
  • border-top-right-radius: define el radio del borde superior derecho.
  • border-bottom-right-radius: define el radio del borde inferior derecho.
  • border-bottom-left-radius: define el radio del borde inferior izquierdo.
  • border-image: define la imagen del borde.
  • border-image-source: define la imagen del borde.
  • border-image-slice: define la porción de la imagen que se utilizará para el borde.
  • border-image-width: define el ancho de la imagen del borde.
  • border-image-outset: define el espacio entre el borde y la imagen.
  • border-image-repeat: define si la imagen se repite o no.

Box Shadow

La propiedad box-shadow nos permite añadir sombras a los elementos de HTML.

La propiedad box-shadow se compone de cuatro valores:

  • offset-x: define el desplazamiento horizontal de la sombra.
  • offset-y: define el desplazamiento vertical de la sombra.
  • blur-radius: define el radio del desenfoque de la sombra.
  • spread-radius: define el radio de la sombra. Si es positivo la sombra se expande, si es negativo la sombra se contrae.
  • color: define el color de la sombra. Si no se especifica este valor se utilizará el color del elemento.
  • inset: define si la sombra se dibuja dentro o fuera del elemento.

Para definir el color de la sombra podemos utilizar cualquier valor de color que hemos visto anteriormente.

Drop Shadow

drop-shadow es una función de filtro que nos permite añadir sombras a los elementos de HTML como imagenes sin fondo.

La función drop-shadow se compone de cuatro valores:

  • offset-x: define el desplazamiento horizontal de la sombra.
  • offset-y: define el desplazamiento vertical de la sombra.
  • blur-radius: define el radio del desenfoque de la sombra.
  • color: define el color de la sombra. Si no se especifica este valor se utilizará el color del elemento.
