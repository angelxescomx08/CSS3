Selectores CSS

Los selectores CSS son una forma de seleccionar los elementos HTML a los que se les aplicará un estilo.

Tipos de selectores

  • Selector universal
  • Selector de tipo
  • Selector de clase
  • Selector de ID
  • Selector de atributo
  • Selectores combinadores

Selector universal

El selector universal selecciona todos los elementos de la página. Se utiliza el asterisco * para indicar que se aplicará a todos los elementos.

* { color: red; }

Selector de tipo

El selector de tipo selecciona todos los elementos de un tipo específico. Se utiliza el nombre del elemento para indicar que se aplicará a todos los elementos de ese tipo.

h1 { color: red; }

Selector de clase

El selector de clase selecciona todos los elementos que tengan un atributo class con el valor especificado. Se utiliza el nombre de la clase precedido por un punto . para indicar que se aplicará a todos los elementos que tengan esa clase.

.clase { color: red; }

Selector de ID

El selector de ID selecciona el elemento que tenga un atributo id con el valor especificado. Se utiliza el nombre del ID precedido por un numeral # para indicar que se aplicará a ese elemento.

#id { color: red; }

Selector de atributo

El selector de atributo selecciona todos los elementos que tengan un atributo con el valor especificado. Se utiliza el nombre del atributo entre corchetes [] para indicar que se aplicará a todos los elementos que tengan ese atributo.

para indicar que se aplicará a todos los elementos que tengan ese atributo.

[atributo] { color: red; }

para indicar que se aplicará a todos los elementos h2 que tengan ese atributo

h2[atributo] { color: red; }

para indicar que se aplicará a todos los elementos que tengan ese atributo con ese valor

[atributo="valor"] { color: red; }

para indicar que se aplicará a todos los elementos que empiecen con ese valor

[atributo^="valor"] { color: red; }

para indicar que se aplicará a todos los elementos que terminen con ese valor

[atributo$="valor"] { color: red; }

para indicar que se aplicará a todos los elementos que contengan ese valor

[atributo*="valor"] { color: red; }

para indicar que se aplicará a todos los elementos que contengan ese valor separado por espacios

[atributo~="valor"] { color: red; }

para indicar que se aplicará a todos los elementos que contengan ese valor separado por guiones

[atributo|="valor"] { color: red; }

Selectores combinadores

Los selectores combinadores son una forma de seleccionar elementos específicos de acuerdo a su relación con otros elementos.

Esto es un selector de descendencia sirve para seleccionar todos los elementos que estén dentro de otro elemento.

div p { color: red; }

Selector de hijo sirve para seleccionar todos los elementos que sean hijos directos de otro elemento por ejemplo todos los elementos p que sean hijos directos de un elemento div.

div > p { color: red; }

Selector de hermano adyacente Sirve para seleccionar todos los elementos que sean hermanos adyacentes de otro elemento por ejemplo todos los elementos p que sean hermanos adyacentes de un elemento div.

div + p { color: red; }

Selector de hermano general Sirve para seleccionar todos los elementos que sean hermanos generales de otro elemento por ejemplo todos los elementos p que sean hermanos generales de un elemento div.

div ~ p { color: red; }
