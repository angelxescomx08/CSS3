Box model

El modelo de cajas es la forma en que se representan los elementos HTML en el navegador. Cada elemento se representa como una caja rectangular.

El modelo de cajas se compone de cuatro partes: el contenido, el padding, el borde y el margen.

Colapso de margenes elementos adyacentes

Cuando dos elementos con margenes verticales se encuentran, el margen más grande se aplica a ambos elementos.

Ejemplo colapso

En el ejemplo se aprecian dos cajas en las que sus margenes chocan entre si, una tiene 30px de margin-bottom y la otra 20px de margin-top, pero el margen que se aplica es de 30px, el más grande de los dos, esto en lugar de los 50px que sería la suma de ambos.

solución colapso

Para soluccionar el colapso se puede convertir la caja uno a un elemento inline-block.

Colapso entre padre e hijo

Cuando un elemento hijo tiene un margen superior el padre absorbe el margen del hijo como si fuera suyo.

Ejemplo colapso

En el ejemplo se aprecian dos cajas una dentro de otra, la caja hijo tiene un margen superior de 30px, pero el padre absorbe el margen del hijo como si fuera suyo, por lo que el margen superior del padre es de 30px en lugar.

Soy el hijo

solución colapso

Para soluccionar el colapso se puede aplicar al padre un padding o un borde esto hará que no se toquen los margenes.

Box sizing

El box sizing es una propiedad que permite cambiar el comportamiento de las cajas, por defecto el box sizing es content-box, esto quiere decir que el ancho y alto que se le da a la caja es solo del contenido, si se le agrega padding o borde el ancho y alto de la caja aumentará.

border-box

Con border-box el ancho y alto que se le da a la caja es del contenido, padding y borde.
