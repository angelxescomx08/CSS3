Espeficidad, herencia y cascada

La espeficidad es la forma en la que el navegador decide que regla de estilo aplicar cuando hay varias reglas que se aplican al mismo elemento.

La espeficidad se calcula sumando los puntos de cada selector. Los selectores se ordenan de mayor a menor espeficidad:

  • Estilos en línea (1000 puntos)
  • IDs (100 puntos)
  • Clases, pseudo-clases, atributos (10 puntos)
  • Elementos, pseudo-elementos (1 punto)

Cascada

La cascada es el orden en el que se aplican las reglas de estilo.

Herencia

La herencia es la forma en la que los elementos hijos heredan las propiedades de los elementos padres.

No todas las propiedades se heredan. Las propiedades que se heredan son aquellas que tienen sentido que se hereden, como el color de texto o el tamaño de fuente.

Se puede forzar a heredar una propiedad con el valor inherit. por ejemplo:

p { color: red; } p span { color: inherit; }
